Тема № 1 - Теги

Мы начали изучать теги HTML, практически все теги "открываются и закрываются" соответствующими знаками, которые знают обсалютно все, поэтому мы их приводить не будем:

  • p - абзац
  • h1,h2... - Заголовки различного порядка
  • style - стилизация компонентов, если можно так выразится
  • body - "тело" самого сайта
  • ul - нумерованный список
  • li - элемент списка
  • ol - ненумерованный (маркерованный) список
  • title - заголовок сайта
  • button - кнопка
  • linc - ссылка на внешний ресурс

Тема № 2 - CSS свойства, компоненты

В этой теме были узучены различные стили для элементов веб-страниц

  1. color - цвет компонента
  2. text-indent - отступ текста от края
  3. text-align - горизонтальное выравнивание
  4. background-color - цвет фона элемнта
  5. width - ширина элемента
  6. height - высота элемента
  7. border - линияч вокруг элемента
  8. font-size - размер шрифта
  9. hover - компонент для стилизации разлчных элементов
  10. font-weight - жирность текста
  11. ... - и некоторые другие
